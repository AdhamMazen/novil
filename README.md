
# Novil Project

This is a simple web project demonstrating the use of HTML, CSS, and JavaScript for a scrolling animation effect.

## Project Structure

- `index.html`: The main HTML file that includes the structure of the webpage.
- `style.css`: The CSS file that contains the styles for the webpage.
- `script.js`: The JavaScript file that handles the scrolling animation effect.
- `image/`: Directory containing the images used in the project.
